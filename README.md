# llama-3.2-3b-alpaca-qlora

Fine-tuned `meta-llama/Llama-3.2-3B-Instruct` on `tatsu-lab/alpaca` using QLoRA for general instruction-following.

> Results and full model card are auto-generated by `scripts/push_to_hub.py` after training and evaluation are complete.

## Quick Start

```bash
uv pip install -r requirements.txt
cp .env.example .env           # add your HF_TOKEN

python scripts/prepare_data.py              # verify dataset
python scripts/evaluate.py --stage before   # baseline metrics
python scripts/train.py                     # ~2.5hrs on RTX 4090
python scripts/evaluate.py --stage after    # post-training metrics
python scripts/evaluate.py --compare        # print comparison table
python scripts/push_to_hub.py               # push to HuggingFace Hub
```

All hyperparameters are in `configs/train_config.yaml`. Update `hub.repo_id` with your HuggingFace username before pushing.
